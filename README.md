### To run the app locally follow next steps:
1. Clone this repo: `$ git clone https://github.com/Katsiaryna-Andrabaila/pianoroll-frontend-challenge`.
2. Install dependencies: `$ npm install`.
3. Start server: `$ npm run dev`.
